From this section you can add an email list. Please note the following:
  • One email list can have multiple Interest Groups associated with it. From this page you have the options to display the interest groups to the users so that they will know what interest groups they are signing up for. If you do not display the interest group, then users will automatically be added the the selected interest groups.
  • If you want an administrator to approve an email before it is added to your list then you should check the option below.
  • You also have the option for users to approve the email before it is added to the database.
  • In addition you have an option for users to be forwarded to any page on the Internet once they sign-up an email from the user interface. Simply fill in the address below and once your users sign-up they will automatically be forwarded to the page you specify.
  • If you check the option for users to review information then when users sign up from your user pages they will be presented with a review page before the information is submitted.
